== Description ==

This plugin automatically searches for ticker symbols in articles and link tickers to company research pages at wallstwatchdog.com. Watchdog Company Pages (beta) provide users with high tech research tools including fully customizable charts, comprehensive fundamental data, trends across the web and professional commentary.

To set it up, just add the plugin to your wordpress installation and activate it.

== Installation ==

1. Upload 'wallstwatchdog-stock-ticker-linker' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. When you write your article, include ticker symbols in parenthesis -- i.g., (AAPL) or (NASDAQ:AAPL)
4. When the article is published, all tickers will be linked to  their research pages on wallstwatchdog.com
